1 Install

sudo add-apt-repository ppa:nilarimogard/webupd8
sudo apt-get update
sudo apt-get install emerald
sudo apt-get install compizconfig-settings-manager

2 Download Theme

Find in Gnome-look or deviantART. I use the littleglass theme.

3 Settings

3.1 Emerald Theme Manager

Load the downloaded theme.theme.

3.2 Compizconfig

Go to Effects/Window Decoration. Type `emerald –replace` in Command.
